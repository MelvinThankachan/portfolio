# Portfolio - Melvin M T

A modern, high-performance portfolio website built with **Next.js 15**, **Tailwind CSS**, and **Magic UI**. 

## üöÄ Features

- **Modern UI/UX**: Built with Magic UI and Shadcn UI components for a premium look and feel.
- **Dynamic Projects Section**: Showcases key projects with direct links to client/server source code.
- **Interactive Dock**: A macOS-style navigation dock for quick access to home, LinkedIn, GitHub, and Resume.
- **Dark/Light Mode**: Seamless theme switching with persistent user preference.
- **SEO Optimized**: Pre-configured with OpenGraph and Twitter meta tags for better reach.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewports.

## üì¶ Getting Started

### Prerequisites

- Node.js 20+ 
- pnpm (or npm/yarn)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/MelvinThankachan/portfolio.git
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Run the development server:
   ```bash
   pnpm dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## üìÑ Data Configuration

The entire portfolio is data-driven. To update the content (name, skills, work, projects), simply modify the `DATA` object in:
